## Features
Ability to select spawn after selecting the character

## Dependencies
- qbr-core
- qbr-multicharacter

## Installation

# Download the script and put it in the [qbr] directory.

# Add the following code to your server.cfg/resouces.cfg

```
ensure qbr-core
ensure qbr-multicharacter
ensure qbr-spawn
```
